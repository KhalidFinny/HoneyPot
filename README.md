# 🍯 HoneyPot 🌿

A delightful **Disney Storybook Expense Tracker** inspired by the cozy adventures of **Winnie the Pooh** and the magical glow of **Rapunzel**. 

*Track your provisions, guard your honey pot, and watch your tales unfold.*

---

## 🎨 Aesthetic & Design

- **Thematic Balance**: Soft Winnie the Pooh honey yellows and natural wood tones, anchored by Rapunzel’s magical sunkissed lanterns and botanical accents.
- **Warm & Cozy**: Opaque **Earth-Linen** backdrops with soft shadows that prevent background clutter, making every card look like a parchment page.
- **Swiss Typography Minimalist Layout**: Neat, dense, compact row layout sizes that avoid overflow so there is absolutely **no horizontal scrolling**.

## 🚀 Features

- **Decide Spending (The Honey Pot)**: Test large expenses against your runway. The honey pot will advise whether it's safe or a thinning warning.
- **Runway Estimator**: See how many months your current provisions will sustain your journey.
- **Budget Meter Control**: A smooth threshold gauge advising you visually once spending limits overlap bounds.
- **Full History Manager Modal**: Edit or Erase items inline flawlessly through immersive sheet triggers absolute speeds.
- **Total Treasure Banner View**: Instant glance headers reading active translation settings setups cleanly.

---

## 🛠 Tech Stack

- **Framework**: React + Vite (Typescript)
- **Styling**: Tailwind CSS
- **Database**: Dexie.js (IndexedDB for local absolute persistence)
- **Icons**: Lucide React

---

## 🌎 Translations

Supports absolute triggers for toggle strings configurations toggling English or Indonesian absolute dictionary datasets concurrently.
